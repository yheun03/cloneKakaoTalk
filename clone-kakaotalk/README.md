# clone-kakaotalk

## 사용 기술 정리

### 프레임워크/라우팅
- Vue 3
- Vue Router 4

### 빌드/번들링
- Vite 6
- @vitejs/plugin-vue

### 라이브러리
- @vuepic/vue-datepicker
- holiday-kr
- vite-svg-loader

### 스타일/폰트
- Sass (SCSS)
- Pretendard 폰트 (프로젝트에 포함된 로컬 폰트)

### 품질/포매팅
- ESLint
- @vue/eslint-config-prettier

### CDN 사용 여부
- 별도 CDN 사용 없음
- 외부 이미지/링크 URL은 데모 데이터로만 사용
